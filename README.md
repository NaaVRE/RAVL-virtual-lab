# Radar Aeroecology Virtual Lab
This virtual lab allows users to easily obtain biological scatters from radar data. 
- High level documentation is available here: https://naavre.net/docs/RAVL/
- A tutorial is available in [NaaVRE](https://beta.naavre.net/vreapp/vl/ravl) in [Virtual Labs/RAVL/Git public/virtual_lab/documentation](./virtual_lab/documentation/RAVL_tutorial.ipynb).
